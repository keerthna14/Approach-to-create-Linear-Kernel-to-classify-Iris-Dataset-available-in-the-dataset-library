🌸 Iris Classification & Pricing System
📌 Overview
A machine learning mini project that classifies iris flowers using petal features and assigns a price based on the predicted species.

⚙️ Technologies
Python
Scikit-learn (SVM)
Tkinter (GUI)
Matplotlib (graph)
🧠 Working
Input: Petal length & width

SVM model predicts species:

Setosa / Versicolor / Virginica
Each species is mapped to a price

Displays model accuracy and decision boundary

🚀 How to Run
Install required libraries
Run the Python file
Enter petal values
Click Predict or view graph
✨ Features
ML-based classification
GUI interface
Accuracy display
Decision boundary visualization
👨‍💻 Conclusion
Demonstrates practical use of machine learning for classification and decision-making with an interactive interface.
